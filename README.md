BMI Calculator

Objective

The objective of this project is to develop a BMI (Body Mass Index) Calculator using Python. The program calculates BMI based on the user's height and weight and displays the corresponding health category.

Steps Performed

1. Collected height and weight from the user.
2. Applied the BMI formula.
3. Calculated the BMI value.
4. Determined the BMI category.
5. Displayed the result to the user.

Tools Used

- Python
- VS Code
- GitHub

Outcome

The project successfully calculates the BMI of a user and displays the appropriate health status such as Underweight, Normal Weight, Overweight, or Obese.

How to Run

1. Open the project in Python.
2. Run the bmi.py file.
3. Enter height and weight when prompted.
4. View the calculated BMI result.
